How to Create an ssh key

For Windows

  1. Open your terminal
  2. Run the Following Command:
    'ssh-keygen -t ed25519 -C "your_email@example.com"'
  3. This will ask you were you want to save it. Just hit enter.
  4. Then it will ask you if you want to add a password. If you decide to enter a password, when you type in your password you will not see it be typed. If you don't want a password, just press enter.
  5. Run the Following Command:
    'clip < ~/.ssh/id_ed25519.pub'
    Running that command will add your ssh key to the clip board.
  6. Using the browser do the following:
    • navigate to github
    • go to settings
    • go to ssh and gpg keys
    • add a new ssh key
    • put your title and paste your key in the big box.
  7. then you are finished

Terminal/Command Prompt Commands

  1. cd : Change Directory
  2. mkdir : Make a New Folder or Directory
  3. cd .. : back out from a folder.
  4. ls : list out all folders and files

Windows Specific

  • dir : list out all files and folders

Mac Specific

  • ls -al : list out all files and folders

Basic Github Command

  • git status : shows what has or hasn't been tracked
  • git add FileName : adds a FileName to your next commit
  • git add . : adds all files in the current directory you are in to your next commit
  • git commit : commits the files without a message NOTE: Do yourself a favor and always add a commit message
  • git commit -m "Your Message Here" : commit the files you are tracking with a message.
  • git push origin BranchName : push your code to BranchName NOTE: We did this with the main branch
    ex. git push origin main
  • git checkout BranchName : switch branches.
  • git checkout -b BranchName : Make a new branch named BranchName
